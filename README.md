# API Security Risk Analysis – JSONPlaceholder

**Tools used:** Postman, Browser DevTools  
**Scope:** `https://jsonplaceholder.typicode.com` endpoints (`/users`, `/posts`, `/comments`)  
**Methodology:**  
- Reviewed OWASP API Security Top 10  
- Performed manual endpoint testing (GET requests)  
- Inspected responses for authentication, authorization, data exposure, rate limiting, and headers  
- Classified risks by severity and business impact  

**Deliverable:** [API Security Risk Analysis Report](./API Security Risk Analysis Report.pdf)  
**Evidence:** See `/screenshots.Zip` folder
